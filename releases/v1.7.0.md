# Poor Man's T-SQL Formatter — Notepad-- Plugin v1.7.0

**Platform:** Windows x64  
**Date:** 2026-06-19

T-SQL formatting plugin for [Notepad--](https://github.com/cxasm/notepad--) (plugin edition), based on [Poor Man's T-SQL Formatter](https://github.com/TaoK/PoorMansTSqlFormatter).

## Download

| Asset | Description |
|-------|-------------|
| `PoorMansTSqlFormatter-Notepad--Plugin-v1.7.0-win-x64.zip` | Prebuilt plugin + formatter (~86 KB) |

## What's included

| File | Description |
|------|-------------|
| `tsqlformatterndd.dll` | Notepad-- plugin (Qt/C++, ~58 KB) |
| `PoorMansTSqlFormatterFmtCli.exe` | T-SQL formatter CLI (stdin/stdout) |
| `PoorMansTSqlFormatterLib.dll` | Core formatting library |
| `INSTALL.txt` | Installation instructions |
| `LICENSE.txt` | GNU AGPL v3 |

## Features

- Format the **entire document** or **selected text only**
- Menu: **Plugins → T-SQL Formatter → Format T-SQL**
- Shortcut: **Ctrl+Alt+F**
- Formatter options: `%APPDATA%\ndd-tsqlformatter\formatter.ini`

## Requirements

- [Notepad--](https://github.com/cxasm/notepad--) with **plugin support** (plugin edition, includes `qmyedit_qt5.dll`)
- [.NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48) runtime (usually preinstalled on Windows 10/11)

## Installation

1. Close Notepad--.
2. Extract the zip.
3. Copy **all files** into your Notepad-- plugin folder:
   ```
   C:\Program Files\Notepad--\plugin\
   ```
   Use an elevated shell if installing under `Program Files`.
4. Restart Notepad--.

## Usage

1. Open a T-SQL file in Notepad--.
2. Select text to format a portion, or leave nothing selected to format the whole file.
3. Press **Ctrl+Alt+F** or use **Plugins → T-SQL Formatter → Format T-SQL**.

## Architecture

```
Notepad--  →  tsqlformatterndd.dll  →  PoorMansTSqlFormatterFmtCli.exe
              (Qt plugin)                (stdin/stdout, .NET 4.8)
                                           ↓
                                    PoorMansTSqlFormatterLib.dll
```

The plugin calls the formatter via Win32 pipes and links dynamically to Notepad--'s `qmyedit_qt5.dll`.
