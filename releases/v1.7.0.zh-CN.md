# Poor Man's T-SQL Formatter — Notepad-- 插件 v1.7.0

**平台：** Windows x64  
**日期：** 2026-06-19

适用于 [Notepad--](https://github.com/cxasm/notepad--)（插件版）的 T-SQL 格式化插件，基于 [Poor Man's T-SQL Formatter](https://github.com/TaoK/PoorMansTSqlFormatter)。

## 下载

| 文件 | 说明 |
|------|------|
| `PoorMansTSqlFormatter-Notepad--Plugin-v1.7.0-win-x64.zip` | 预编译插件 + 格式化器（约 86 KB） |

## 包内文件

| 文件 | 说明 |
|------|------|
| `tsqlformatterndd.dll` | Notepad-- 插件（Qt/C++，约 58 KB） |
| `PoorMansTSqlFormatterFmtCli.exe` | T-SQL 格式化命令行（stdin/stdout） |
| `PoorMansTSqlFormatterLib.dll` | 格式化核心库 |
| `INSTALL.txt` | 安装说明 |
| `LICENSE.txt` | GNU AGPL v3 |

## 功能

- **无选区**：格式化整篇文档；**有选区**：仅格式化选中部分
- 菜单：**Plugins → T-SQL Formatter → Format T-SQL**
- 快捷键：**Ctrl+Alt+F**
- 选项文件：`%APPDATA%\ndd-tsqlformatter\formatter.ini`

## 系统要求

- 已安装 [Notepad--](https://github.com/cxasm/notepad--) **插件版**（含 `qmyedit_qt5.dll`）
- [.NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48) 运行时（Windows 10/11 通常已预装）

## 安装步骤

1. 退出 Notepad--。
2. 解压 zip 包。
3. 将**全部文件**复制到 Notepad-- 插件目录：
   ```
   C:\Program Files\Notepad--\plugin\
   ```
   若安装在 `Program Files` 下，需使用管理员权限。
4. 重新启动 Notepad--。

## 使用方法

1. 在 Notepad-- 中打开 T-SQL 文件。
2. 选中部分文本则只格式化选区；不选中则格式化全文。
3. 按 **Ctrl+Alt+F**，或通过 **Plugins → T-SQL Formatter → Format T-SQL**。

## 架构

```
Notepad--  →  tsqlformatterndd.dll  →  PoorMansTSqlFormatterFmtCli.exe
              (Qt 插件)                  (stdin/stdout, .NET 4.8)
                                           ↓
                                    PoorMansTSqlFormatterLib.dll
```

插件通过 Win32 管道调用格式化器，并与 Notepad-- 自带的 `qmyedit_qt5.dll` 动态链接。
